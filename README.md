# simple-games-in-c++

A collection of classic games built using C++ that run directly in the terminal. This repository demonstrates the use of fundamental programming concepts, logic, and data structures in C++.

🎮 Games Included
Tic-Tac-Toe (X-O): A classic two-player strategy game.

Rock, Paper, Scissors: A simple game against the computer.

Bingo.

🚀 Features
C++ Powered: Uses Advanced STL and efficient logic.

Terminal Interface: Lightweight and runs in any CLI environment.

Interactive Gameplay: User-friendly input handling.

🛠️ Technologies Used
Language: C++

Tools: VS Code & GitHub
